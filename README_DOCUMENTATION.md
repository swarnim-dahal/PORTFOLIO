## ğŸ¨ Color Customization Points

All colors are CSS variables. Change them in `App.css`:

```css
--neon-cyan: #00d9ff;      /* Main accent */
--neon-purple: #d946ef;    /* Secondary */
--neon-green: #39ff14;     /* Success */
--dark-bg: #0a0e27;        /* Background */
--dark-secondary: #1a1f3a; /* Secondary bg */
--text-primary: #e0e0e0;   /* Main text */
--text-secondary: #a0a0a0; /* Muted text */
```
